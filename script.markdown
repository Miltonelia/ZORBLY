# ZORBLY AGENT 🤖⚡

The first decentralized multi-agent AI platform for creators on Solana. A single prompt does everything a creator needs. Agents evolve forever with community votes.

## ✨ Features

- **One Prompt Does All**: Single input field triggers content creation, token launch, social posting, and community management
- **5 Powerful Agents**:
  - 🎨 Content Agent: Generates memes, videos & captions
  - 🚀 Launch Agent: Deploys memecoins on Pump.fun
  - 📱 Social Agent: Auto-posts to X, Telegram, YouTube, TikTok
  - 👥 Community Agent: Creates Tally forms & manages signups
  - 🧬 Mutation Agent: Evolves based on community votes every 24h
- **Solana Integration**: Phantom wallet support, $ZORBLY token payments & staking
- **Community Governance**: Vote for agent mutations and new features
- **Live Dashboard**: Real-time stats, engagement metrics, activity feed
- **Telegram Mini-App**: Full support for Telegram Web App SDK

## 🚀 Quick Start

### Option 1: Deploy to Vercel (Recommended)

1. Fork this repository
2. Go to [Vercel](https://vercel.com)
3. Click "New Project"
4. Import your forked repository
5. Click "Deploy"
6. Done! Your app will be live at `https://your-project.vercel.app`

### Option 2: Deploy to Netlify

1. Fork this repository
2. Go to [Netlify](https://netlify.com)
3. Click "Add new site" → "Import an existing project"
4. Select your repository
5. Build settings:
   - Build command: (leave empty)
   - Publish directory: `.`
6. Click "Deploy site"
